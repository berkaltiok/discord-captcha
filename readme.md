# Project Setup

Always use <a href="https://yarnpkg.com/">yarn</a> to avoid conflicting .lock file issues.

## Install

First, install all packages with `yarn`:

```bash
yarn
```

## Development

For starting to development, use this command:

```bash
yarn start
```
